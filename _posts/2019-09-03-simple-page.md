## Simple page

```
<!DOCTYPE html>
<html>
  <head>
    <meta http-equiv="Content-Type" content="text/html; charset=utf-8">
    <title>Hello HTML5 World</title>
  </head>
  <body>
    <h1>Hello HTML5</h1>
    <p>Welcome to the future of markup!</p>
  </body>
</html>
```

---

## Ordered list

```
<!DOCTYPE html>
<html>
  <head>
    <meta http-equiv="Content-Type" content="text/html; charset=utf-8">
    <title>Ordered list</title>
  </head>
  <body>
    <ol>
     <li>First item</li>
     <li>Second item</li>
     <li>Third item</li>
    </ol>
  </body>
</html>
```
